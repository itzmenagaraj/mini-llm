A language model learns by predicting the next token.
A token can be a character, a word, or a subword.
The model reads input tokens and predicts the next token.
During training, the model compares prediction with target.
The difference is called loss.
Lower loss means better prediction.

A transformer uses self attention.
Self attention helps each token look at previous tokens.
Query means what the token is looking for.
Key means what information a token has.
Value means what information a token passes forward.
Attention scores decide which tokens are important.

Token embedding converts token IDs into vectors.
Position embedding tells the model the order of tokens.
The transformer block processes embeddings.
The feed forward layer improves each token representation.
The output layer produces logits.
Softmax converts logits into probabilities.
The next token is selected from the probability distribution.

Question: What is a language model?
Answer: A language model predicts the next token based on previous tokens.

Question: What is self attention?
Answer: Self attention allows tokens to focus on other relevant tokens in the input sequence.

Question: What is loss?
Answer: Loss measures how wrong the model prediction is during training.

Question: What is an embedding?
Answer: An embedding is a vector representation of a token.

Question: What is a transformer?
Answer: A transformer is a neural network architecture that uses attention to process sequences.
